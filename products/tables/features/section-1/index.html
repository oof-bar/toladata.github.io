Data Collection and Aggregation.

Tola Tables is a tool for all types of data collection methods. Designed initially to solve the problem of duplicate, disaggregated or broken data from mobile data collection platforms such as Ona and iFormbuilder, Tola Tables can now integrate data from multiple platforms and provide merge, auto-export, data cleaning, editing, sharing and analysis features. It also provides the data service back end for other Tola tools such as Tola Workflow.
